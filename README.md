# GrantSignal Landing Page

Static landing page for testing demand for a manual non-dilutive funding research service.

## Files

- `index.html`: landing page markup
- `styles.css`: page styles
- `app.js`: example-preview interactivity

## Deployment

This repo is configured for GitHub Pages via GitHub Actions.

Once pushed, the site should publish automatically from the `main` branch workflow.

## Before Taking Real Orders

Replace the current placeholder order flow with:

- a real payment link for `Standard ($25)`
- a real payment link for `Priority ($39)`
- an intake form asking for topic, location, applicant type, and email
- delivery wording that matches actual service hours

## Suggested Manual Fulfillment Flow

1. Buyer pays
2. Buyer submits intake form
3. Report is created manually
4. Finished report is delivered by email as PDF or DOCX
